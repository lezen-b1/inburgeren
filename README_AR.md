# NT2 Lezen B1 - موقع تدريب موثق

هذه نسخة تدريب غير رسمية لقراءة NT2 B1. الهدف هو التدريب على النصوص والأسئلة الرسمية المرفقة داخل المشروع، مع حفظ التقدم محليًا على جهاز الطالب.

## المحتوى المهيكل

- 2020: 36 سؤالًا رسميًا كاملًا مع مفتاح الإجابة.
- 2021: 37 سؤالًا رسميًا كاملًا مع مفتاح الإجابة.
- 2022: 35 سؤالًا رسميًا كاملًا مع مفتاح الإجابة.
- 2023: 35 سؤالًا رسميًا كاملًا مع مفتاح الإجابة.
- 2024: 35 سؤالًا رسميًا كاملًا مع مفتاح الإجابة.
- 2025: جزئي تفاعليًا، 19 سؤالًا من أصل 35. لا تُضاف أسئلة أو اختيارات أو أدلة غير موثقة.

إجمالي الأسئلة الرسمية الكاملة 2020-2024: **178 سؤالًا**.
إجمالي المسائل المهيكلة مع 2025: **197 مسألة**.

## الأدلة التفصيلية

بعض الأسئلة تعتمد حاليًا على مفتاح الإجابة الرسمي فقط `answer-key-only`. عندما لا يكون موضع الدليل التفصيلي موثقًا داخل البيانات، يعرض الموقع رسالة صادقة بدل صندوق دليل فارغ.

راجع `CONTENT_EVIDENCE_COVERAGE_AR.md` لمعرفة التغطية حسب السنة.

## الصفحات الرئيسية

- **النماذج:** اختيار السنة وبدء نموذج تفاعلي حيث تتوفر البيانات، أو فتح PDF.
- **التدريب:** جلسات قصيرة مع النص قبل السؤال ومراجعة بعد كل خمسة أسئلة.
- **المكتبة:** 90 تحليلًا لإعادة الصياغة.
- **التقدم:** سجل التدريب، نتائج النماذج الكاملة، الأخطاء والكلمات غير المفهومة.
- **المصادر:** أسماء الملفات والتنبيه إلى أن الموقع غير رسمي وأن حقوق الملفات تعود لأصحابها.

## النسخ الاحتياطية

التقدم محفوظ في IndexedDB داخل المتصفح. من صفحة الإعدادات يمكن:

- تصدير نسخة JSON للتقدم والجلسات.
- معاينة ملف النسخة قبل الاستيراد.
- استيراد ملف صالح بعد التحقق الكامل.

لا تُمسح البيانات الحالية إلا بعد نجاح التحقق من ملف النسخة الاحتياطية.

## التشغيل محليًا

```bash
npm ci --no-audit --no-fund
npm run dev
```

## الفحص والبناء

```bash
npm run check
npm run build
```

في هذه البيئة المرفقة لم يكن أمر `npm` متاحًا من PATH، لذلك استُخدمت أدوات Node المضمّنة لتشغيل TypeScript وVitest وVite.

## النشر على GitHub Pages

ملف `.github/workflows/deploy.yml` لم يُعدّل في هذه الجولة. إعداد `base` يبقى `/inburgeren/` ويعمل التطبيق عبر `HashRouter`.

## حقوق الاستخدام

ملفات PDF قد تخضع لحقوق نشر المصدر الأصلي. هذه النسخة لا تدّعي ملكية الملفات الرسمية، وعلى المستخدم التحقق من شروط إعادة النشر والاستخدام.

## تحديث فتح ملفات PDF

أزرار «فتح» تستخدم الآن صفحة قراءة مستقلة مبنية على PDF.js داخل مسار `#/pdf`، مع استثناء ملفات PDF من Navigation Fallback الخاص بالـPWA. أزيل قسم «نماذج تدريبية حديثة» من صفحة النماذج لأنه لا يحتوي تدريبًا تفاعليًا مهيكلًا.
